# top-pop-server
express server top-pop

## To install packages 
* yarn install

## To run server
* yarn run serve

## To use server
* Open browser by your choice
* enter localhost:3000

# To get all tracks
* localhost:3000/tracks

# To get one track
* localhost:3000/tracks/id
